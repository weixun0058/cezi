## 签号计算算法

### 原理
诸葛神算中的签号计算是基于汉字笔画数的算法，用来将用户输入的三个汉字转换为1-383之间的签号，然后根据签号查询对应的卦象信息。

### 实现方法
算法在`utils.py`的`cale_character_count`函数中实现，主要步骤如下：

1. 获取三个汉字的康熙笔画数
2. 对每个笔画数取模10，得到个位数（如果结果为0则使用1）
3. 将三个个位数组合成一个三位数
4. 对该三位数取模384，得到1-383之间的签号

### 代码分析
```python
def cale_character_count(num1, num2, num3):
    """
    计算三个数字对应的签号
    Args:
        num1: 第一个汉字的康熙笔画数
        num2: 第二个汉字的康熙笔画数
        num3: 第三个汉字的康熙笔画数
    Returns:
        返回1-383之间的签号
    """
    # 方法1：简单相加后取余
    last_digit1 = num1 % 10 or 1
    last_digit2 = num2 % 10 or 1
    last_digit3 = num3 % 10 or 1
    
    # 使用383作为最大签号
    new_number = last_digit1 * 100 + last_digit2 * 10 + last_digit3
    
    result = new_number % 384
    
    return result
```

### 汉字笔画数获取
汉字的笔画数通过`database.py`中的`get_stroke_count`函数获取，该函数优先从本地数据库查询，如果查询不到则从在线汉典网获取并缓存结果。

## 干支与纳音五行计算

### 原理
干支纪年是中国传统历法中的重要概念，由天干和地支组合而成。纳音五行是对干支组合的进一步解释，将不同的干支组合映射到五行（金、木、水、火、土）的具体属性上。

### 干支计算
干支由lunar_python库计算，主要通过以下方法获取：
- 年干支：`lunar.getYearInGanZhi()`
- 月干支：`lunar.getMonthInGanZhi()`
- 日干支：`lunar.getDayInGanZhi()`
- 时干支：`lunar.getTimeInGanZhi()`

### 纳音五行计算
纳音五行计算在JavaScript中实现（`lunar_date_handler.js`），通过查表的方式将干支组合映射到对应的纳音五行：

## 节气计算算法

### 原理
节气是中国传统历法中反映季节变化的重要概念，一年分为24个节气。节气的计算基于天文学原理，主要是通过太阳在黄道上的位置来确定。

### 实现方法
本项目使用lunar_python库进行节气计算，该库基于寿星天文历算法实现了高精度的节气计算。

## 黄历数据生成算法

### 原理
黄历数据包含多种传统历法信息，如宜忌、神煞、彭祖百忌等。这些数据基于农历日期、干支、节气等多个因素计算得出。

### 实现方法
系统通过`huangli.py`中的`_generate_huangli_data`函数生成黄历数据，主要步骤如下：

1. 解析输入的公历日期
2. 转换为农历日期对象
3. 获取干支、生肖、节气等基础信息
4. 计算宜忌、神煞、彭祖百忌等传统黄历元素
5. 组合成完整的黄历数据

### 代码分析
生成黄历数据的核心代码如下：

## 八字命理分析算法

### 原理
八字命理分析基于中国传统命理学，通过分析一个人出生时的年、月、日、时四柱天干地支，推测其命运走向。

### 实现方法
系统通过`lunming.py`中的方法实现八字命理分析，包括：

1. 收集用户的姓名、性别、出生日期和时辰信息
2. 使用lunar_python库计算干支和纳音五行
3. 生成分析提示词
4. P调用LLM（大语言模型）进行分析

### 关键代码分析
生成八字分析请求的核心代码：

```python
def analyze_bazi(self, name, gender, birth_date, birth_time):
    # 生成提示词
    prompt = self.generate_prompt(name, gender, birth_date, birth_time)
    
    # 调用AI模型进行流式输出
    client = OpenAI(
        api_key="API密钥",
        base_url="API地址"
    )
    
    stream = client.chat.completions.create(
        model="模型名称",
        messages=[
            {'role': 'user', 'content': prompt}
        ],
        temperature=0.7,
        stream=True
    )
    
    # 处理流式输出
    for chunk in stream:
        if hasattr(chunk.choices[0], 'delta') and hasattr(chunk.choices[0].delta, 'content'):
            if chunk.choices[0].delta.content:
                content = chunk.choices[0].delta.content
                # 一个字符一个字符地返回
                for char in content:
                    yield char
```

## 总结

诸葛神算项目的核心算法涵盖了中国传统命理学和历法学的多个方面，包括：

1. **签号计算算法**：将汉字笔画转换为签号，实现占卜功能
2. **干支与纳音五行计算**：提供完整的天干地支和纳音五行分析
3. **节气计算算法**：基于寿星历算法的高精度节气计算
4. **黄历数据生成算法**：全面的传统黄历信息生成
5. **八字命理分析算法**：结合传统命理学和现代AI技术的个性化分析

这些算法相互配合，共同构建了一个既保留传统文化精髓，又结合现代技术的命理预测系统。系统的核心价值在于将复杂的传统历法和命理学知识通过算法形式化，使其能够在现代计算机系统中得到应用和传承。
