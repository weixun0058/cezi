# 诸葛神算V2 - 公农历日期转换器

这是一个交互式的公历与农历日期转换工具，提供以下功能：

- 公历日期选择器（使用HTML5的date输入控件）
- 农历日期选择器（年、月、日分开的下拉选择）
- 公历与农历日期的双向转换
- 支持1900-2100年的日期范围
- 支持农历闰月显示
- 响应式设计，适配移动设备

## 技术栈

- HTML5
- CSS3
- JavaScript
- 农历转换库

## 使用方法

1. 打开index.html文件
2. 在公历日期选择器中选择日期，农历日期会自动更新
3. 或者在农历选择器中选择年、月、日，公历日期会自动更新

## 项目结构

- `index.html`: 主页面
- `css/style.css`: 样式文件
- `js/calendar.js`: 日期转换逻辑
- `js/lunar.js`: 农历转换库
