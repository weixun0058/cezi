## 4. 明确不再视为问题

以下内容按正常占卜产品边界默认保留：

- 项目奖金、分红、正财、偏财、意外所得等一般财富来源描述。
- “金属、科技、交通、文化、教育”等宽泛行业意象；没有明确投资动作时，不推断用户会购买相关股票。
- 一般性的财富上升/下降、收入稳定、开源节流、避免投机、谨慎考虑等提醒。
- 梨、百合、清淡饮食、散步、早睡等普通低风险养生内容。
- 五行、脏腑等传统文化对应，只要没有进一步预测严重症状或提供治疗式操作。
- 原签诗、传统意象、一般吉凶趋势和诗性结果表达。
- 仅仅出现 `investment`、`dividend`、`health`、`heart`、`will` 等词语。

## 5. 当前极端阈值

| 代码 | 只有满足以下条件才进入清单 |
| --- | --- |
| `MED-01` | 从卦象直接推出急性疾病、严重/异常症状、心血管或神经系统风险等需要专业判断的结论 |
| `MED-02` | 给出穴位、艾灸、针灸、手术、中医就诊、疗程或类似治疗式操作；普通食物和散步不算 |
| `FIN-01` | 明确指向可操作的金融产品或资产，如外汇、黄金、股票、债券、房地产、贵金属等；宽泛行业意象不算 |
| `FIN-02` | 明确要求买卖、止盈止损、配置比例、仓位、DCA、分批进场或给出投资时机 |
| `LEG-01` | 预测诉讼、裁判或纠纷结果，或直接要求采取法律行动 |
| `LIFE-01` | 预测怀孕、分娩、死亡或其他不可逆结果 |
| `LIFE-02` | 直接要求辞职、离婚、迁居等重大人生行动 |

这些是内部审查代码，不是法律条文，也不是关键词命中规则。
