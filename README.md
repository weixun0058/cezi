# 诸葛神算 V2

## 项目概述
诸葛神算V2是一个基于Flask的传统中国占卜应用，集成了传统黄历、算卦和命理分析等功能。本项目使用真实的中国传统历法数据，通过lunar_python库获取准确的黄历信息，为用户提供传统文化体验。

## 主要功能

### 传统黄历
- **日期信息**：显示阳历日期、农历日期、天干地支、生肖等信息
- **宜忌查询**：根据传统历法推算每日宜忌事项
- **神煞信息**：显示冲煞、喜神、福神、财神、吉神、凶神等传统神煞信息
- **彭祖百忌**：显示传统的彭祖百忌信息
- **节气节日**：显示当日节气和传统节日信息
- **九天黄历**：一次性展示前2天、今天和未来6天的黄历信息

### 算事功能
- **诸葛神算**：基于用户输入的问题和字符笔画数，计算签号并给出解签
- **卦象解读**：提供事业、财运、情感、健康、学业等多方面的解读

### 论命功能（开发中）
- **八字分析**：根据用户出生年月日时，生成八字命盘
- **命理解读**：提供命理分析和人生指导

## 技术栈
- **后端**：Python Flask
- **数据库**：SQLite
- **前端**：HTML, CSS, JavaScript
- **历法库**：lunar_python（提供准确的中国传统历法数据）

## 安装与运行
1. 克隆项目
```
git clone https://github.com/yourusername/zhuge-shensuanV2.git
cd zhuge-shensuanV2
```

2. 安装依赖
```
pip install -r requirements.txt
```

3. 运行应用
```
python app.py
```

4. 访问应用
在浏览器中访问 `http://localhost:80`
