# 诸葛神算 V2

诸葛神算是一个基于传统占卜方法与现代技术相结合的网络应用，提供黄历查询、卜卦算事和论命功能。

## 项目概述

本项目模拟传统的占卜方式，结合现代技术，为用户提供一种娱乐性的命运预测体验。项目包含三个主要功能模块：

1. **黄历系统**：提供每日黄历信息，包括宜忌、吉凶方位、节气、节日等内容。
2. **算事系统**：用户通过输入汉字，系统计算笔画数，生成卦象，并提供解读。
3. **论命系统**：（开发中）将提供基于生辰八字的命盘分析。

## 技术栈

- **后端**：Python Flask
- **前端**：HTML, CSS, JavaScript
- **数据库**：SQLite
- **第三方库**：lunar-python（黄历计算）

## 功能特点

### 黄历系统

- 每日黄历查询
- 宜忌事项推荐
- 吉祥方位、颜色、数字提示
- 节气与传统节日信息
- 一周黄历预览

### 算事系统

- 汉字笔画计算
- 卦象生成与解读
- 多维度运势分析（事业、财运、情感、健康、学业等）

### 论命系统（开发中）

- 生辰八字分析
- 命盘解读
- 流年运势预测

## 安装与运行

1. 克隆项目到本地
2. 安装依赖：`pip install -r requirements.txt`
3. 运行应用：`python app.py`
4. 访问 `http://localhost:80` 使用应用

## 使用说明

### 黄历功能

1. 访问主页 `/`
2. 查看当日黄历信息，包括宜忌、吉凶方位等
3. 使用日期选择器或前后按钮切换日期
4. 查看一周黄历预览，点击任意日期查看详情

### 算事功能

1. 点击主页的"算事"链接或直接访问 `/suanshi` 路径
2. 在输入框中输入三个汉字
3. 点击"开始测算"按钮
4. 系统会计算汉字笔画，生成卦象
5. 查看卦象解读和多维度运势分析

### 论命功能

1. 点击主页的"论命"链接或直接访问 `/lunming` 路径
2. 此功能正在开发中，敬请期待

## 注意事项

- 本应用仅供娱乐，请勿将结果作为重要决策依据
- 黄历数据基于传统历法计算，仅供参考
- 最佳浏览体验请使用现代浏览器
